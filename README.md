# Duolingo Landing Page Clone

## 📚 Project Overview

This project is a **team assignment** developed during the **Web Developer course at Develhope**.  
The goal is to recreate the **Duolingo website landing page** as closely as possible, focusing on structure, layout, and visual design.

At this stage of the project, we are working **only with HTML and CSS**, without JavaScript or external frameworks.

This exercise helps us strengthen our front-end fundamentals by practicing:

- Semantic HTML structure
- CSS styling and layout
- Responsive design principles
- Team collaboration using Git and GitHub

---

## 🎯 Project Goals

- Reproduce the look and feel of the Duolingo homepage
- Build a clean and organized HTML structure
- Apply CSS for styling, spacing, positioning, and responsiveness
- Practice collaborative development in a team environment

---

## 🛠️ Technologies Used

- **HTML5**
- **CSS3**
- **Git**
- **GitHub**

---

## 👥 Team Project

This is a **collaborative team project**, where each member contributes to different sections and features of the page.

Through this project, we are also practicing:

- Branch management
- Pull requests
- Merge conflict resolution
- Code organization and teamwork
